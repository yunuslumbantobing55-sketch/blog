Singkatnya, GitHub bekerja seperti "Google Docs" tapi khusus untuk kode. Berikut adalah beberapa poin utamanya: 
GitHub

Penyimpanan Kode (Hosting): GitHub menjadi tempat untuk menaruh proyek coding Anda dalam wadah yang disebut Repository.
Kolaborasi Tim: Memungkinkan banyak orang bekerja pada proyek yang sama secara bersamaan tanpa saling menimpa pekerjaan satu sama lain.
Version Control: Menggunakan teknologi bernama Git untuk mencatat setiap perubahan yang dilakukan. Jika ada kesalahan, Anda bisa dengan mudah kembali ke versi kode sebelumnya.
Portofolio: Developer sering menggunakan profil GitHub mereka sebagai portofolio untuk menunjukkan karya mereka kepada calon perekrut.
